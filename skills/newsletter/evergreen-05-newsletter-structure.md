# Evergreen Skill 05: Newsletter Structure

Goal: Create a predictable, readable structure.

Standard structure:
1. Subject line and preview text
2. Opening hook
3. Main story or lead section
4. Supporting sections (2-4 items)
5. CTA or engagement prompt
6. Closing signature

Formatting:
- Use short paragraphs.
- Use subheads to break sections.
- Include bullets for lists or steps.
